## 3. 一句话简介：这个作品是什么 *

打通个人多音乐平台的 3D 歌词音乐播放器，把网易云、本地曲库与 Navidrome 等来源汇到同一套沉浸式舞台里听歌。

## 4. 一句话亮点：你觉得它最酷的地方 *

最酷的是「视觉 + 互通 + 智能匹配」一体：全屏歌词像文字 PV，多平台曲库可一起用，本地歌还能自动补歌词与封面。
