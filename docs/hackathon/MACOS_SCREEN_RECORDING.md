# macOS 自带录屏完整指南

快速参考：如何使用 Mac 自带工具录制 Lyra 演示视频。

---

## 🎯 推荐方案：Cmd + Shift + 5

这是 macOS 最简单、最快速的录屏方式。

### 快速步骤

1. **按快捷键**：`Cmd + Shift + 5`
2. **选择录制模式**：录制整个屏幕
3. **设置选项**（点击"选项"）：
   - 保存位置：桌面
   - 麦克风：关闭
   - 显示鼠标点击：关闭（可选）
4. **点击"录制"**
5. **停止录制**：点击菜单栏的停止按钮
6. **文件自动保存**到桌面

## 🎬 完整录制流程

### 1. 准备环境

```bash
# 构建生产版本并启动
./quick-record.sh

# 或手动执行
npm run build
npm run preview
```

### 2. 设置录制环境

- 开启勿扰模式：`Cmd + Option + D`（或系统设置）
- 关闭所有通知
- 清理桌面
- 纯色壁纸

### 3. 准备浏览器

- 访问：http://localhost:4173
- 全屏模式：`Cmd + Ctrl + F`
- 准备好演示内容

### 4. 开始录制

1. 按 `Cmd + Shift + 5`
2. 选择"录制整个屏幕"
3. 点击"录制"按钮
4. 按照演示脚本进行 3 分钟演示
5. 点击菜单栏停止按钮

## 🎥 QuickTime Player 方法

如果你需要更多控制和编辑功能：

### 步骤

1. 打开 **QuickTime Player**
2. **文件** → **新建屏幕录制**
3. 点击红色录制按钮
4. 选择录制方式：
   - **点击任意位置**：录制整个屏幕
   - **拖动选择区域**：录制特定部分
5. 点击"开始录制"
6. 演示 3 分钟内容
7. 菜单栏点击停止
8. **文件** → **存储**（选择位置和名称）

### 编辑功能

QuickTime 提供基础编辑：
- **修剪**：`Cmd + T`（裁剪开头结尾）
- **分割**：`Cmd + Y`
- **旋转**：编辑菜单
- **导出**：不同质量选项

## 🔧 常见问题

### Q: 录制的文件在哪里？
**A**: 默认保存在桌面。可以在"选项"中更改保存位置。

### Q: 如何提高录制质量？
**A**: macOS 自动使用最高质量。确保：
- 使用 Retina 显示器
- 关闭省电模式
- 关闭其他占用资源的应用

### Q: 文件太大怎么办？
**A**: 使用 HandBrake 或 iMovie 压缩：
- HandBrake（免费）：`brew install --cask handbrake`
- 目标码率：5-8 Mbps
- 保持 1080p 60fps

### Q: 如何添加背景音乐？
**A**: 使用 iMovie：
1. 导入录制文件
2. 添加音频轨道
3. 调整音量（30-40%）
4. 导出

### Q: 录制时卡顿怎么办？
**A**: 
- 关闭其他应用
- 降低浏览器分辨率
- 关闭浏览器扩展
- 使用 Safari（性能更好）
